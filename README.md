# Isaac Newton Website

A user-friendly informational website showcasing the life, achievements, and contributions of Sir Isaac Newton. This project was created to practice frontend web development using HTML and CSS.

## Features
- Clean and beginner-friendly UI
- Biography and achievements sections
- External reference links
- Simple navigation bar
- Responsive webpage structure

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose of the Project
This project was built as part of a web development workshop to improve understanding of:
- Website structure
- Styling with CSS
- Basic frontend design
- Project organization using Git and GitHub

## Future Improvements
- Improve UI/UX
- Make the website fully responsive
- Add animations and better navigation
- Deploy online
